# **App Name**: Billetera Clara

## Core Features:

- Registro de ingresos: Permite a los usuarios registrar sus ingresos especificando la fuente y la cantidad.
- Registro de gastos: Permite a los usuarios registrar sus gastos, categorizándolos según categorías personalizables.
- Visualización de datos: Muestra gráficos interactivos que representan la distribución de los ingresos y gastos a lo largo del tiempo.
- Entrada de datos sin conexión: Permite a los usuarios ingresar datos incluso sin conexión a Internet, sincronizando los datos una vez que se restablece la conexión.
- Almacenamiento de Datos: Almacena de forma segura los datos de ingresos y gastos utilizando la base de datos pgAdmin4.
- Asignación de fondos: Permite a los usuarios destinar fondos a gastos específicos, descontando automáticamente los gastos realizados de ese fondo asignado.

## Style Guidelines:

- Color primario: Verde claro (#A7F3D0) para representar la estabilidad financiera y el crecimiento.
- Color de fondo: Blanco (#FFFFFF) para una apariencia limpia y organizada.
- Color de acento: Un tono ligeramente más oscuro de verde (#74E2B2) para resaltar elementos importantes y botones de acción.
- Fuente para el cuerpo del texto y titulares: 'PT Sans', una fuente sans-serif humanista, para facilitar la lectura y proporcionar un estilo moderno.
- Utilizar iconos claros y minimalistas para representar las categorías de ingresos y gastos, facilitando la identificación visual.
- Diseño limpio y estructurado con una navegación intuitiva para facilitar la entrada de datos y la visualización de la información financiera.
- Animaciones suaves y sutiles al cambiar entre las diferentes secciones y al actualizar los gráficos para mejorar la experiencia del usuario.
